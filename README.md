# Starbound Console Command Guide

Guide to getting achievements in Starbound via console commands.

## The Hopeful [x]

```
/startquest '"human_mission1"'
```
```
/completequest human_mission1
```

## Augmented Reality [x]

```
/spawnitem damageaugment2 1
```
